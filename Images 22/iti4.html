Computer Operator and Programming Assistant

Govt. ITI Deoni

Course Overview

COPA is a one-year vocational training program under NCVT. It provides essential computer skills, programming basics, data entry, web design, and IT knowledge.

Vision & Mission

Vision

To develop digitally skilled youth for modern technology.

Mission

  • Provide computer and programming training
  • Develop IT and employability skills
  • Encourage entrepreneurship

Course Details

  • Duration: 1 Year
  • Eligibility: 10th Pass
  • Age: 14+
  • Fees: ₹1770 / ₹970

Syllabus

  • Computer Basics
  • MS Word, Excel, PowerPoint
  • HTML, CSS, JavaScript
  • MySQL Database
  • Python / Java
  • Cyber Security

Career Opportunities

  • Computer Operator
  • Data Entry Operator
  • Web Designer
  • IT Support
